## 우리는 프로그램의 성능을 정확히 알 수 있는가?

- 입력크기
- 하드웨어 성능
- 운영체제 성능
- 컴파일러 최적화
- 비동기 로직

프로그램의 성능을 정확히 파악하는 것은 불가능하다. 때문에 추상적으로나마 성능을 파악하기 위한 수단으로 만든 것이 빅오 표기법이다.

## 빅오 표기법

O(1) < O(log n) < O(n) < O(n log n) < O(n^2) < O(2^n) < O(n!)
